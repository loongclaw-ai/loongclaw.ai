# 运行时引擎

运行时引擎是 LoongClaw 的核心，管理它如何处理信息、记住过去的交互以及使用工具。

## 在本节中：

- **[记忆](/docs/runtime/memory)**：了解 LoongClaw 如何持久化状态和召回信息。
- **[会话](/docs/runtime/sessions)**：了解如何独立管理多个聊天上下文。
- **[工具](/docs/runtime/tools)**：使用内置的 Shell、文件和 Web 工具扩展 LoongClaw 的功能。
