# 连接性

连接性支柱关注 LoongClaw 如何与外部世界通信——无论是与 AI 模型还是与用户。

## 在本节中：

- **[LLM 提供商](/docs/connectivity/providers)**：配置与 OpenAI、Anthropic、DeepSeek、Kimi、Gemini 等的连接。
- **[通信渠道](/docs/connectivity/channels)**：设置 Telegram、飞书 (Lark) 和 CLI 等通信渠道。
