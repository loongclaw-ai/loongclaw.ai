# Features

LoongClaw provides rich features to help you build powerful AI applications.

## Chat

Powerful interactive chat with multi-session management and persistent storage.

- Multi-session isolation
- SQLite persistence
- Sliding window memory
- History viewing

[Learn more →](/docs/features/chat)

## Tool System

Built-in powerful tool runtime supporting safe execution of system commands and file operations.

- Shell command execution
- File read/write operations
- Security whitelist mechanism
- Capability authorization

[Learn more →](/docs/features/tools)

## Channel Integration

Support for multiple communication channels to meet different scenario requirements.

- CLI command-line interface
- Telegram Bot
- Feishu application

[Learn more →](/docs/features/channels)

## Spec Workflow

Advanced workflow definition format using JSON to describe complex execution flows.

- Workflow orchestration
- Permission control
- Audit tracing
- Lifecycle management

[Learn more →](/docs/features/spec)
