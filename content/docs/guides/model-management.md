# Model Management

Manage your local models with simple commands.

## List Models

```bash
claw list
```

## Pull a Model

```bash
claw pull literati-7b
```

## Remove a Model

```bash
claw rm literati-7b
```

## Copy a Model

```bash
claw cp literati-7b my-model
```
